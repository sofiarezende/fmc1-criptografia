# fmc1-criptografia
trabalho da unidade 2 da disciplina de FMC 1 Mensagem final: requisicao falha por timeout. A parte mais importante do trabalho foi feito inspirado pela lógica da potenciação modular do livro Matemática Discreta e Suas Aplicações da autora Kenneth H. Rosen. O algoritmo está na seção 3.6, página 227 do livro. O algoritmo basicamente analisa cada digito binário de D_binario e se o dígito for 1, iremos multiplicar x pela potência (e ajustar o módulo para permanecermos com números pequenos, para evitar problemas de overflow). Note que a potência, para cada interação ela se eleva ao quadrado. Logo, estamos basicamente multiplicando x pelas potências em que D_binario é 1, dividido pelo módulo e refazendo essa etapa com o resto. O algoritmo irá fornecer um número, mas para decodificar toda a mensagem, precisei repetir o algoritmo 19 vezes com o restante dos numeros. Aí basta separar esses números encontrados em blocos de dois e analisar na tabela dada pelo professor o significado de cada bloco. Assim, encontrei a mensagem final: requisicao falha por timeout.
